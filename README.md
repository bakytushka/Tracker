# Назначение

Приложение помогает пользователям формировать полезные привычки и контролировать их выполнение.

## Цели приложения

- Контроль привычек по дням недели
- Просмотр прогресса по привычкам

## Краткое описание

Приложение состоит из карточек-трекеров, которые создает пользователь. Он может указать название, категорию и задать расписание. Также можно выбрать эмодзи и цвет, чтобы отличать карточки друг от друга.

Карточки отсортированы по категориям. Пользователь может искать их с помощью поиска и фильтровать.

С помощью календаря пользователь может посмотреть, какие привычки у него запланированы на конкретный день.

В приложении есть статистика, которая отражает успешные показатели пользователя, его прогресс и средние значения.

# Функциональные требования

## Онбординг
- Экран с заставкой, заголовком, вторичным текстом и кнопкой «Вот это технологии» для перехода на главный экран.

## Создание карточки привычки
- Возможность создать трекер для привычки или нерегулярного события.
- Поля для ввода названия, выбора категории, эмодзи, цвета и настройки расписания.
- Кнопки «Отменить» и «Создать». Кнопка «Создать» активна только при заполнении всех полей.

## Просмотр главного экрана
- Отображение всех созданных трекеров на выбранную дату.
- Поля для поиска, фильтрации и добавления новых привычек.
- Модальные окна для редактирования и удаления карточек.

## Редактирование и удаление категории
- Долгое нажатие на категорию для редактирования или удаления.
- Модальные окна для подтверждения действий.

## Просмотр статистики
- Отображение успешных показателей, прогресса и средних значений.
- Показатели: лучший период, идеальные дни, завершенные трекеры и среднее значение.

## Темная тема
- Автоматическая смена темной темы в зависимости от системных настроек.

# Нефункциональные требования
- Поддержка iPhone X и выше, адаптация под iPhone SE; минимальная версия iOS – 13.4.
- Использование стандартного шрифта iOS – SF Pro.
